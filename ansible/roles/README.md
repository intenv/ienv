
# Иерархия доступных ролей
## Стандартная структура каталогов и файлов для роли
Структура взята с [Best Practices | http://docs.ansible.com/ansible/latest/playbooks_best_practices.html]
```
├── roles
│   ├── common
│   │   ├── defaults
│   │   │   └── main.yml
│   │   ├── files
│   │   ├── handlers
│   │   │   └── main.yml
│   │   ├── library
│   │   ├── lookup_plugins
│   │   ├── meta
│   │   │   └── main.yml
│   │   ├── module_utils
│   │   ├── README.md
│   │   ├── tasks
│   │   │   └── main.yml
│   │   ├── templates
│   │   └── vars
│   │       └── main.yml
│   └── README.md
```
